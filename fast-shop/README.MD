# FastShop OpenSource eCommerce Template
## Stack
1. Django
2. MySql
3. Jinja2
4. Vue.js
5. Django-ninja


## How to run
1. Clone
2. pip install -r requirements.txt
3. python manage.py makemigrations
4. python manage.py migrate
5. python manage.py createsuperuser
6. python manage.py runserver
